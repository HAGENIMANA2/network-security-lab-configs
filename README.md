# Network Security Lab Configurations

## About This Repository

This repository contains demonstration configurations, templates, and procedures for network security engineering tasks. All content is based on my practical experience in:

- Firewall configuration (Cisco ASA, iptables)
- VPN setup (IPsec, OpenVPN)
- Access control lists (ACLs) and port security
- VLAN design and network segmentation
- Vulnerability assessment and security hardening
- Security monitoring with SNMP and IPS/IDS
- Incident response planning
- Security documentation
